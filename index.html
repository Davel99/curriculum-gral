Descripción de lo realizado

Se realizó la aplicación para el Departamento de contratos legados en la ciudad de Tampico durante un periodo total de seis meses. A continuación se detallan las características implementadas:

  • 1) Base de datos para el almacenamiento de información
  • 2) Sistema de tipos de usuarios con contraseñas protegidas con hashing
  • 3) Sistema para usuarios administradores sobre gestión de usuarios normales
  • 4) Gestión (adición, eliminación o modificaición) de datos de la empresa referentes a las zonas y sus clientes
  • 5) Sistema de almacenamiento y lectura de archivos en servidor
  • 6) Comparación de archivos CSV con miles de registros y clasificación de datos
  • 7) Interfaz de filtros para elección de datos deseados
  • 8) Generación de archivos excel listos para descarga

Se utilizó composer para la administración de paquetes. El principal paquete usado fue "phpoffice" para la generación de archivos excel. Además, se trabajó con bootstrap para el diseño de las interfaces. También se implementó una API con PHP y Jquery (con Ajax) para la comunicación entre el servidor y el cliente en tiempo real.
